# Educational Data Mining for Student Performance

## 📌 Project Overview
This project focuses on analyzing student performance data to identify key factors affecting academic results.

## 🎯 Objective
- Analyze student data
- Identify performance patterns
- Provide insights for improvement

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- Excel
- Data Visualization (Matplotlib / Power BI)

## 📊 Key Analysis
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Performance Trends

## 📈 Key Insights
- Attendance and study time impact student performance
- Parental education plays a role in results
- Certain subjects show consistent low scores
